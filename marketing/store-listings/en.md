## Full description (4000 char)

ERPAIO is an AI assistant that lets you talk to your ERP systems (Nebim V3 first, more coming) in natural Turkish.

— Ask "What's our top selling product this month?" — system writes SQL, returns answer.
— Say "What was last year's revenue?" — table appears in seconds.
— Stock levels, due dates, customer-level sales — query in seconds.

ANOMALY DETECTION
Cron-based monitoring scans your data hourly. Sales drops, critical stock, overdue payments — anomalies caught automatically. High-priority alerts hit WhatsApp + push notification.

FEATURES
• Turkish natural language queries (Claude Sonnet 4 AI)
• 6 prebuilt anomaly metrics (hourly + daily)
• WhatsApp + email + push notifications
• Chat history, CSV export, edit-rerun SQL
• Multi-tenant — each company in its own data sandbox
• AES-256-GCM encrypted ERP credentials
• KVKK + GDPR compliant

SECURITY
Your ERP data is NOT stored on our side; only processed during queries. Credentials encrypted at rest with AES-256-GCM. SQL validator, prompt injection protection, rate limiting, OWASP Top 10 hardening.

LEARNING SYSTEM
With your 👍/👎 feedback the system improves over time. QueryCache returns repeat queries in sub-500ms.

SUPPORTED ERPs
• Nebim V3 (official support)
• SAP, Oracle, Dynamics 365 (coming soon)
• Generic SQL Server / PostgreSQL / MySQL — limited support

PLANS
Starter — individual use
Pro — team, multi-user, unlimited queries
Enterprise — on-prem agent, dedicated support

Currently in pilot with active customers. Details: erpaio.vercel.app
